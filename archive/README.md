# Archive

This directory preserves historical material moved out of the active project
root during the 2026-04-29 cleanup.

Do not delete this directory without a specific preservation decision.

Layout:

- `legacy_papers/`: old paper drafts and submission variants.
- `legacy_duplicates/`: older duplicate paper folders kept for traceability.
- `legacy_submissions/`: reserved for older submission packages.
- `legacy_chroma_dbs/`: local Chroma vector stores, ignored by git.
- `legacy_results/`: reserved for retired result bundles.
- `old_zips/`: old zip snapshots, ignored by git.
- `old_logs/`: old run logs and Kaggle working snapshots, ignored by git.
- `miscellaneous/`: older notes that do not belong in active docs.
